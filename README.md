# 🧊 Cubo Rubik Interactivo

Proyecto interactivo donde el cursor genera cubos estilo Rubik y cambia el color del fondo dinámicamente.

---

## 🚀 Tecnologías

- HTML5
- CSS3
- JavaScript

---

## ✨ Características

- 🧊 Cubos tipo Rubik generados con el mouse
- 🖱️ Interacción en tiempo real
- 🌈 Cambio dinámico de colores de fondo
- ⚡ Efectos visuales creativos
- 🎨 Experiencia interactiva

---

## 📂 Estructura del proyecto

```
cubo-rubik-interactivo/
│
├── cubo.html
├── style.css
├── script.js
```

---

## ▶️ Cómo usar

1. Descarga el proyecto
2. Abre `cubo.html` en tu navegador
3. Mueve el mouse y haz clic para ver los efectos
